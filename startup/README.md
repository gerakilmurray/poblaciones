# poblaciones
Herramienta para la publicación colaborativa de datos espaciales de población

En esta carpeta se encuentran los elementos necesarios para inicializarse en el uso de Poblaciones.

Las guías permiten:

1. Instalar una versión en un sitio productivo.

2. Obtener y utilizar los fuentes para armar un entorno de desarrollo.

3. Convertir un entorno de desarrollo en una versión instalable.
